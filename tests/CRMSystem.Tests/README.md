# CRMSystem.Tests

Unit and integration tests for the CRM System.

## Frameworks

- **xUnit** — test framework
- **Moq** — mocking
- **FluentAssertions** — readable assertions
- **EF Core InMemory** — for repository/integration tests

## Structure (planned)

- `Services/` — unit tests for business services
- `Repositories/` — integration tests using in-memory DB
- `Validation/` — validator tests
